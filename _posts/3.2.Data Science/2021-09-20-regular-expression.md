# Regular Expression(RegEx)

### Special characters

- .(dot) : matches any character
- \w : word character
- \s : empty space
- \+ : resulting RE to match 1 or more repetitions
- \* : resulting RE to match 0 or more repetitions
- ^(caret) : beginning of string
- $(dollar) : end of string
