##2.1텍스트 줄 바꿈
```
기본적인 텍스트 표기 방식
마트다운은 줄바꿈을 인식하지 않는다. 

줄바꿈을 하기위해서는 라인 끝에 스페이스를 두번  
표기해야 한다. 
여러가지 강조 표시가 존재한다. 
첫번째로 *single asterisks*  
두번째로 _single undersocres_  
세번째로 **double asterisks**  
네번째로 __double underscores__  
다섯번째로 ~~cancel line~~
